# 周报/OKR 自动生成器

> SKU: PD-003 · 售价 ¥9.9

---

## 解决什么问题

- 每周五下午花 1-2 小时憋周报
- 老板看不懂 / 觉得你没干活
- OKR 复盘没思路，凑字数

## 怎么用

**Step 1**：把本周做的事情，按"做了什么 + 结果数据 + 遇到的坑"结构列 5-10 条
**Step 2**：把这段文字喂给 AI，附上 prompt 包
**Step 3**：AI 给你 3 个版本的周报：简版（3 行）/ 标准版（一页）/ 详细版（带数据图表）
**Step 4**：挑一个复制粘贴到飞书/钉钉/企业微信

## Prompt 核心

```
你是我公司资深助理。基于我本周的工作碎片，生成一份让老板一眼看懂的周报。

我的周报要满足 3 个原则：
1. 老板能在 30 秒内 get 我这周干了啥
2. 必须有数据（哪怕是定性数据）
3. 必须有下周计划

本周工作碎片：
{{粘贴你的 bullet points}}

输出 3 个版本：
- 简版（3 行）
- 标准版（一页 PPT）
- 详细版（带数据可视化建议）
```
