## 套路 1：数字 + 痛点 + 解法

**模板**：{数字}个{痛点}，{解法}！

**示例**：
- 5 个 Excel 神技，会 3 个就是高手！
- 7 天瘦 5 斤的早餐公式，懒人也能瘦！

**Prompt**：
```
你是小红书爆款标题专家。请基于我提供的主题，生成 20 个标题，要求：
1. 使用"数字+痛点+解法"公式（如"5 个 XX，3 个就是高手"）
2. 字数 ≤ 20 字
3. 加入 1 个情绪词（震惊/哭了/跪了/神了/绝）
4. 避免标题党但要抓眼球
5. 标注预期互动率（高/中/低）

主题：{{你的主题}}
```

## 套路 2：反差 + 颠覆认知

**模板**：{大众认知}？其实{颠覆真相}！

**示例**：
- 月入 3 万很难？其实是你没学会这 1 招！
- 减肥不能吃米饭？医生：那是 30 年前的观点！

**Prompt**：
```
你是反常识标题大师。基于主题，找出 3 个大众认知误区，并产出 20 条颠覆性标题：
- 结构：{大众认知}？其实{真相}！
- 必带反转词：其实/真相是/医生说/内行人才知道
- 控制在 18 字内

主题：{{你的主题}}
```

## 套路 3：人设 + 福利 + 紧迫感

**模板**：{身份}整理的{资源}，{限时}领取！

**示例**：
- 10 年 HR 整理的简历模板，24h 后删！
- 北大妈妈整理的 0-6 岁绘本清单，仅限前 100 名！

**Prompt**：
```
你是人设包装高手。基于主题，构造一个可信身份 + 高价值资源 + 紧迫感的标题：
- 身份要具体（年份/头衔/学历/职级）
- 资源要具体（数量/类型/价值）
- 紧迫感：限时/限量/即将删除

主题：{{你的主题}}
```

## 套路 4：对比 + 转折 + 痛点

**模板**：{A 状态}→{B 状态}，只因我做了{1 件事}

**示例**：
- 月薪 5k → 月薪 3w，我做对了这 1 件事！
- 60 分 → 90 分论文，我只改了 3 个地方！

**Prompt**：
```
你是故事型标题专家。基于主题，构造"前后对比 + 1 个改变"的标题：
- A 状态：要具体可量化
- B 状态：要比 A 状态好 2-10 倍
- 1 件事：要足够具体（不能说"努力"，要说"每天 30 分钟读这本书"）

主题：{{你的主题}}
```

## 套路 5：提问 + 戳痛点 + 答案预告

**模板**：{为什么}{痛点}？{答案就在文中}！

**示例**：
- 为什么你化妆总是显脏？答案在这 3 个细节！
- 为什么孩子越来越不爱说话？这 1 招家长必看！

**Prompt**：
```
你是戳痛点提问专家。基于主题，生成 20 条"为什么+痛点+答案预告"标题：
- 痛点要扎心（戳中读者不敢说出口的难处）
- 答案要可执行（不能是"看了就知道"这种废话）
- 结尾带"！"或"？"增强感染力

主题：{{你的主题}}
```

## 套路 6：场景 + 身份 + 行动

**模板**：{具体场景}的{身份}，{正在做某事}！

**示例**：
- 通勤地铁上的打工人，都在偷偷做这 1 件事！
- 双 11 当晚的宝妈，手机里一定有这 3 个 App！

**Prompt**：
```
你是场景化标题专家。基于主题，生成 20 条"具体场景+身份+行为"标题：
- 场景要具体到时间/地点/天气/动作（不能是"生活中"这种泛词）
- 身份要明确（年龄/职业/状态）
- 行为要可观察（"在偷偷做 X"比"在思考"更具体）

主题：{{你的主题}}
```

## 套路 7：盘点 + 排序 + 价值

**模板**：{年份}最值得{动作}的{数量}个{对象}，第 X 个太香了！

**示例**：
- 2024 最值得买的 5 个家居好物，第 3 个真的香！
- 程序员的 10 个提效神器，第 7 个我天天用！

**Prompt**：
```
你是盘点型标题专家。基于主题，生成 20 条"年份+数量+排序惊喜"标题：
- 必带年份（今年）
- 数量：5/10/15 最佳
- 必须有 1 个明显的"高光推荐"（第 X 个太香了/第 X 个绝绝子）
- 高光位置随机，避免每次都在第 1 或最后

主题：{{你的主题}}
```
